# 🚀 **AURA CLIENT - DEPLOYMENT INSTRUCTIONS**

## 📋 **ЧТО ГОТОВО ДЛЯ ДЕПЛОЯ:**

### ✅ **API Сервер**
- **Node.js** сервер на `simple-server.js`
- **JWT аутентификация** с HWID привязкой
- **API эндпоинты** для лаунчера
- **Static файлы** для скачивания

### ✅ **Лаунчер**
- **C++** лаунчер с WebView2
- **Анти-отладочные** защиты
- **HWID генерация**
- **API интеграция**

### ✅ **Клиент**
- **Forge 1.16.5** мод ClientGuard
- **Обфускация** через ProGuard
- **Проверка модов** и защита

## 🎮 **КАК ЭТО БУДЕТ РАБОТАТЬ:**

### **Для пользователей:**
1. **Заходят на сайт**: `https://aura-client-sites.vercel.app`
2. **Скачивают лаунчер**: `/downloads/AuraLauncher.exe`
3. **Запускают лаунчер**
4. **Вводят логин/пароль**: `test / test123`
5. **Лаунчер качает клиент**: `/downloads/AuraClient.jar`
6. **Запускается Minecraft** с защитами

### **Защиты работают:**
- ✅ **HWID привязка** - один аккаунт = одно устройство
- ✅ **Проверка модов** - только разрешенные моды
- ✅ **Анти-отладка** - нельзя взломать лаунчер
- ✅ **SHA-256 верификация** - целостность файлов
- ✅ **JWT токены** - безопасная авторизация

---

## 🛡️ **УРОВНИ ЗАЩИТЫ:**

### **Level 1: Сеть**
- **HTTPS/TLS** шифрование
- **JWT токены** с expiry
- **Rate limiting** от DDoS
- **CORS** защита

### **Level 2: API**
- **HWID проверка** при входе
- **Manifest подпись** для файлов
- **Whitelist** модов
- **Heartbeat** проверки

### **Level 3: Лаунчер**
- **Анти-отладка** (PEB, тайминги)
- **Обфускация** строк
- **Проверка целостности** файлов
- **Безопасный HTTP** клиент

### **Level 4: Клиент**
- **ClientGuard мод** с приоритетом 1000
- **Проверка лаунчера** через JVM args
- **Сканирование модов** с хэшами
- **Runtime tampering detection**
