## 🎮 **ПОЛНЫЙ ФЛОУ СИСТЕМЫ:**

### **Пользователь:**
1. **Скачивает лаунчер** → `/downloads/AuraLauncher.exe`
2. **Запускает лаунчер** → WebView2 interface
3. **Вводит данные** → API проверка
4. **Качает клиент** → `/downloads/AuraClient.jar`
5. **Запускает игру** → Forge + ClientGuard
6. **Играет безопасно** → Анти-чит защиты

### **Система:**
1. **HWID привязка** → 1 аккаунт = 1 устройство
2. **Проверка модов** → Только разрешенные
3. **SHA-256 верификация** → Целостность файлов
4. **Анти-отладка** → Защита от взлома
5. **JWT токены** → Безопасная сессия
